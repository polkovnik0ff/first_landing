## Шаги

1. Прочитай все файлы памяти из `C:\Users\keman\.claude\projects\f--projects-first-landing\memory\`
2. Выполни `git -C f:/projects/first_landing log --oneline -10` чтобы увидеть последние коммиты
3. Выполни `git -C f:/projects/first_landing status` чтобы увидеть незакоммиченные изменения

## Запиши запись в лог

Добавь новую запись в начало файла `C:\Users\keman\.claude\projects\f--projects-first-landing\memory\activity_log.md` в следующем формате:

```

### Что сделано (последние коммиты)
- <список последних коммитов с кратким описанием что изменилось>

### Незакоммиченные изменения
- <если есть — перечислить, иначе написать "нет">

### Активные планы
- <список незавершённых задач из памяти>

### Состояние проекта
- <2-3 предложения о текущем состоянии сайта и что ждёт следующим>

---
```

Если файл `activity_log.md` не существует — создай его с заголовком `# Activity Log — first_landing` перед первой записью.

Не делай ничего кроме записи в лог. Не коммить, не меняй код.
